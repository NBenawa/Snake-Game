# 🐍 Classic Snake Game

A simple and fun implementation of the classic Snake Game built with Python. Control the snake, collect food, grow longer, and try to achieve the highest score without colliding with the walls or yourself.

## 🎮 Features

* Smooth snake movement
* Random food spawning
* Score tracking
* Game Over screen
* Keyboard controls
* Lightweight and beginner-friendly code

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed:

```bash
python --version
```

Python 3.8 or higher is recommended.

### Installation

1. Clone this repository:

```bash
git clone https://github.com/yourusername/snake-game.git
```

2. Navigate to the project directory:

```bash
cd snake-game
```

3. Install dependencies (if any):

```bash
pip install -r requirements.txt
```

### Run the Game

```bash
python snake.py
```

## 🎯 Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |
| ESC | Exit Game  |

## 🏆 Scoring

* Each food item collected increases your score.
* The snake grows longer after eating food.
* The game ends if the snake collides with:

  * The game boundaries
  * Its own body
